Boost your Bricks websites width Tailwind CSS. Build faster, easier, and better!

== Description ==

Flowtitude increases your efficiency in building websites with Bricks and Tailwind CSS thanks to contextual help in Bricks Builder panels.

== Installation ==

1. Download the plugin's zip file.

2. Go to Plugins > Add New in your WordPress admin. Click "Add New" button, then "Upload Plugin" button, then "Choose File", browse to and select the plugin's zip file.

3. Activate the plugin.
